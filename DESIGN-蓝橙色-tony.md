# 暗蓝色设计系统

> 从配色样例图中提取的暗蓝色主题配色方案。
> 深色底色 + 暖金色点缀 — 深邃、沉稳、专业的安全工具气质。

## 提取来源

`配色样例图.jpg` — 深色 UI 设计样图，以深海军蓝为基底，暖金色为唯一强调色。

## 色板

### 背景层级（从深到浅）

| 名称 | 色值 | 色块 | 用途 |
|------|-------|------|------|
| Obsidian Deep | `#040d1a` | ██ | 最深底色，用于极深层级或 hover 叠加 |
| Obsidian Base | `#05152c` | ██ | 页面主背景 (12.9%) |
| Obsidian Alt | `#06172b` | ██ | 背景变体、footer (12%) |
| Obsidian Mid | `#0a1825` | ██ | 二级背景 (6%) |
| Surface Dark | `#0f2636` | ██ | 卡片、面板、表面背景色 |
| Surface Mid | `#0d2736` | ██ | 卡片变体、高亮表面 |

### 边框与分割线

| 名称 | 色值 | 用途 |
|------|-------|------|
| Border | `#1a2a3a` | 卡片边框、分割线 |

### 强调色 — 暖金色

| 名称 | 色值 | 用途 |
|------|-------|------|
| Gold Accent | `#ee9038` | 主强调色、CTA 按钮、重要链接 |
| Gold Hover | `#f3a13e` | 悬停状态、亮部强调 |
| Gold Deep | `#d47828` | 暗部强调、阴影 |

### 文本色

| 名称 | 色值 | 用途 |
|------|-------|------|
| Text Primary | `#e8edf3` | 标题、导航、正文强调 |
| Text Body | `#9ca9b9` | 正文、描述文本 |
| Text Secondary | `#5a6f82` | 辅助文本、说明文字 |
| Text Muted | `#3c4f60` | 禁用状态、占位符 |

| 名称 | 色值 | 用途 |
|------|-------|------|
| Success | `#4ade80` | 成功状态 |
| Error | `#ef4444` | 错误、警告 |

## CSS 自定义属性

```css
:root {
  /* 背景层级 */
  --bg-deep: #040d1a;
  --bg-base: #05152c;
  --bg-alt: #06172b;
  --bg-mid: #0a1825;
  --bg-surface-dark: #0f2636;
  --bg-surface-mid: #0d2736;

  /* 边框 */
  --color-border: #1a2a3a;

  /* 强调色 — 暖金 */
  --color-gold: #ee9038;
  --color-gold-hover: #f3a13e;
  --color-gold-deep: #d47828;

  /* 文本 */
  --text-primary: #e8edf3;
  --text-body: #9ca9b9;
  --text-secondary: #5a6f82;
  --text-muted: #3c4f60;

  /* 功能色 */
  --color-success: #4ade80;
  --color-error: #ef4444;
}
```

## 设计规则

### Do
- 使用 `#05152c` 作为页面主背景
- 使用 `#0f2636` 作为卡片/表面背景
- CTA 按钮使用 `#ee9038` 金色并带有微弱发光
- 链接和重要交互元素使用暖金色
- 正文使用 `#9ca9b9`，标题使用 `#e8edf3`
- 分割线使用 `#1a2a3a`，保持低对比度

### Don't
- 不要使用亮色渐变破坏深色氛围
- 不要在非交互元素上使用金色
- 不要使用纯白色 (`#ffffff`) 作为正文文本
- 不要让任何元素有强烈的投影（最多使用微光）
- 不要使用蓝色作为强调色（保持金色为唯一强调色）
